R.G.S Trip Ticket System

Github Repository

A final project for both Information Systems and Web Programming, R.G.S Trip Ticket System is a delivery management website that tracks delivery trip tickets sent in by drivers and to be approved by administrators. The project was built on CSS, HTML, and Javascript, and uses external applications and websites such as XAMPP and phpMyAdmin.
